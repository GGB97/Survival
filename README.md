# Survival
> 유니티 3D로 만든 간단한 생존게임 틀?


## 사용/테스트 방법
  * 시작 씬 ->SampleScene
  * 조작 방법
    * 이동 : WASD
    * 점프 : SPACE
    * 상호작용 : E
    * 시점 전환 : 마우스
    * 공격 : 마우스 좌클릭
    
## 시연 영상
  * [링크](https://youtu.be/y43_tgUCZWI)

## 개발 환경
  * Unity 2022.3.2f1

## 팀원

* 길기범
    * 구현 기능 : 플레이어(이동, 점프, 공격), 낮/밤 구현, 아이템과의 상호작용, 아이템 장착과 모션, 자원 채취, 적(곰), 발걸음 소리 및 뮤직존
